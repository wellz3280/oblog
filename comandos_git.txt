comandos git

git config --global user.name "seu nome"
git config --global user.email seuemail@provedor.com

git confg --local user.name "seu nome" - para ser usado apenas em um projeto específico

git init 

git status 

git add . - adiciona os arquivos
gi add "nome do arquivo" -  adciona 1 arquivo especifico

git commit -m "comita e deixa menssagem"

git log - mostra tudo q ja foi comitado
git log --online - mostra as alterações em uma linha
git log -p - mostra as alteraçoes do commit
mais comandos do git log https://devhints.io/git-log

Caso queira que o git ignore um arquivo. Crie um arquivo .gitignore.
No arquivo .gitignore coloque nome do arquivo.extensão. 
Se tiver em um diretorio diretorio/arquivo.extensão
Use o git add no .gitignore e use o git commit -m
